== Description ==

This plugin does default wordpress widget but without links. It is only shows name and comment.

Features

This plugin add last comments widget your widgets area. 
Different is; default last comments widget have commenter URL but this widget show only name and last comment text. 
It cant show url or author web address. 



[WordPress Tema](http://www.sinanisler.com/)


== Installation ==


1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Now you can use widget on widgets section.
